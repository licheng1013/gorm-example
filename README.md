- 创建数据库 t_gorm
- 导入项目的sql文件到该数据库中
- **数据库的配置在项目的config/mysql下，你需要手动修改下**
- [https://gitee.com/licheng1013/gorm-example/blob/master/config/mysql.go](https://gitee.com/licheng1013/gorm-example/blob/master/config/mysql.go)
